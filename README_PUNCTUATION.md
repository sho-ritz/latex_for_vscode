# 句読点自動変換設定

このプロジェクトでは、LaTeXファイル（`.tex`）を編集する際に、「、。」と入力すると自動的に「，．」に変換されます。

## セットアップ

1. VSCode拡張機能「Replace On」をインストール
   - VSCodeを開くと、拡張機能のインストールを促す通知が表示されます
   - または、拡張機能マーケットプレイスで「Replace On」を検索してインストール

2. `.tex`ファイルを開いて、「、」や「。」を入力すると、自動的に「，」「．」に変換されます

## 動作確認

`.tex`ファイルを開いて：
- 「、」と入力 → 自動的に「，」に変換されます
- 「。」と入力 → 自動的に「．」に変換されます

## 注意事項

- この設定は、このワークスペース（プロジェクト）内でのみ有効です
- LaTeXファイル（`.tex`）を編集している時のみ動作します
- コメント行（`%`で始まる行）でも変換されます
